# README

## Message-me app

## This app is created for learning and training purposes

## used technologies:

- Ruby v 2.7.4
- Ruby on rails v 5.2.6
- SQlite as Database
- Semantic-ui as CSS framework

## To run this app locally:

```
git clone https://github.com/NDragneelL9/message-me.git
```

```
bundle install
```

```
rails db:migrate
```

```
rails server
```
